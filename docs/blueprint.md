# **App Name**: Page Highlighter

## Core Features:

- Search Term Input: Allows the user to input a search term via a popup form.
- Search Mode Selection: Toggle between exact search and regular expression modes.
- Text Highlighting: Dynamically highlights all matching text on the current webpage in yellow.
- Match Counter: Displays a count of the number of matches found on the page.

## Style Guidelines:

- Primary color: White or light gray for the background.
- Secondary color: Dark gray for text and borders.
- Accent: A shade of blue (#3498db) for interactive elements like buttons and toggles.
- Clean and readable font.
- Simple and intuitive popup layout.
- Subtle fade-in effect for the highlighted text.
